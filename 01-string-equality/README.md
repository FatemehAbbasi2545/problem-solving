# string-equality
Problem Description: A program that reads two string values ​​from input and determines whether the two strings are equal or not. Then returns the result to the main program.
The condition for equality of strings is:
1- Their number of characters must be equal.
2- Every character that exists in the first string must be repeated the same number of times in the second string.
3- If a character does not exist in the first string, it must not exist in the second string either.
